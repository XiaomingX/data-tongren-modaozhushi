# 仙侠奇遇

蓝忘机站在讲台上，淡淡地说道：“处理妖邪有三步：首先是化解它的执念与怨气，其次是镇压，最后才是彻底消灭。反正得遵循这个顺序，千万别搞错了。” 

大家长舒了一口气，心里暗自庆幸，幸好是蓝忘机在这里发话，不然轮到他们，搞不好就漏了一步或者是顺序搞错了。蓝启仁满意地点了点头：“说得不错！无论修行还是为人，得扎扎实实。如果只因为某些小成就就自满，那迟早会自讨苦吃。” 

魏无羡挑了挑眉，偷偷瞄了蓝忘机一眼，心想：“老头这是打算让我好好享受一场学习。” 

他开口说道：“我有疑问。”  

蓝启仁冷冷一笑：“说。”  

魏无羡接着说：“虽然说化解执念是第一，但是如果执念是想要复仇，那该怎么办？”  

蓝忘机插口：“所以度化为主，镇压为辅，有必要时才灭绝。”  

魏无羡微微一笑：“听起来不错。”停顿了一下，他又提出新的思路：“这名刽子手死了变成了凶尸，不如挖出那些被他杀的人，激起他们的怨气，让他们一起对付这个凶尸……” 

蓝忘机终于转过头来，眉头皱得紧紧的，神色显得很冷漠。蓝启仁不得不用力抖动胡须，“你这是什么天真的想法！” 

蓝启仁愤怒地站起身：“我们处理妖邪，就是为了化解，让他们得以安息！你竟然想着激怒怨气，这完全不是大道之理！”  

魏无羡回嘴道：“我觉得有些事情根本就没法化解，那就不如利用一下怨气。大禹治水都知道堵水是下策，疏导才是上策！”一边说着，他一边躲开了蓝启仁扔过来的书，继续喋喋不休的辩论。  

蓝启仁又是一声怒喝：“那我问你，怎么保证这些怨气能为你所用，不反噬他人？”  

魏无羡嘴角一扬：“我还没想好！” 

这下蓝启仁火了，吼道：“你若想到了，那仙门百家都留不下你！滚！”  

魏无羡心里一喜，连忙就退开了。  

于是他在云深不知处闲逛，待到半日，终于在高高的墙上找到了他，那一幕，宛如天神下凡。魏无羡正坐在墙头，悠闲地叼着草，双手撑着腮，一条腿翘着，另一条腿随意晃动。下面的人指着他哈哈大笑：“魏兄！你真有意思，居然真的跑了！”  

“他脸色铁青，估计还没反应过来！”  

魏无羡吐了一口草：“我说的都是实话嘛，让我滚就滚呗！”  

聂怀桑好奇地问：“蓝老头咋对你这么严厉啊？”  

江澄撇嘴：“活该，他问的那些问题，谁敢这么答？真以为在自己家里轻松！”  

魏无羡没在意：“我本来就想和他好好理论一下，结果他不理我。”  

聂怀桑迟疑片刻道：“其实魏兄说的也有道理，要是能利用怨气，岂不是更方便？修炼灵气那么难，何不试试呢？”  

江澄提醒：“说得再好听也别走那邪路。”  

魏无羡笑着回答：“我可不是傻蛋，怎么会走那种路！”  

“喂，想不想趁今晚没宵禁，跟我去打山鸡？”  

江澄严厉回应：“打山鸡？这里哪来的山鸡！你快去抄《雅正集》吧，蓝启仁让我转告你，让你抄《上义篇》三遍。”  

魏无羡撅撅嘴：“三遍？我抄一遍飞升就好了。又不是蓝家人，干嘛抄他们家训？”  

聂怀桑急忙跳出来：“我帮你抄！我帮你抄！”  

魏无羡好奇地问：“你想干嘛？”  

“其实，蓝老头有个坏毛病，他……”   

这时，蓝忘机从远处走来，正好听见。魏无羡转头，看到蓝忘机背着剑，远远地望着他们，心里一颤。  

“糟了，蓝忘机来了！”  

“快闭嘴！”  

这时，蓝忘机踏着轻盈的步伐，就像一抹白色的云影，淡淡地走开。魏无羡气鼓鼓：“为什么他总是这样对我？”  

“他难道不算好看吗？”聂怀桑小声说，“只是这情绪好像得修炼修炼。”  

三天后，魏无羡终于明白蓝启仁的坏毛病是什么。蓝启仁讲课冗长得不可思议，结果每次都要默写。魏无羡听得那叫一个头昏眼花，而聂怀桑则担心：“魏兄，求求你，别再出乱子了！”  

可惜，魏无羡总是风风火火，带坏了周围。于是他再次被罚，惩罚更是重得可怕。  

“你不能出去，得跟我一起抄《礼则篇》！”聂怀桑无奈地摇头，魏无羡只好郁闷地走进蓝家的藏书阁。  

在藏书阁里，蓝忘机正安静地抄写，文字优雅的让人沉醉。魏无羡心里想着：“虽然气氛有点紧张，但他真是个美男子。”  

“真想让那些姑娘来看看，如果这个人总是沉默，哪怕长得再帅，也没用。”  

在这个充满仙尘和魔溪的奇幻世界里，魏无羡与蓝忘机错综复杂的关系，也在悄然间酝酿着新的故事……
