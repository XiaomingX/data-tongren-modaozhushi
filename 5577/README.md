# 修仙围猎

在一个阳光明媚的日子里，金夫人在广场上居高临下，脸上满是怒气：“子勋！”，这一声怒喝让魏无羡的笑容瞬间消失，周围的气氛也变得紧张起来。

“家教？”魏无羡慢慢转身，一脸轻松，嘴角微微上扬，“你们口中所谓的‘邪魔歪道’其实也能让我风光无限。”他的话如同一道惊雷，震惊在场所有人，个个瞪大了眼睛，几乎不敢相信自己耳朵。

这时，金子勋再也忍耐不住了，愤怒地吼道：“魏无羡！你不过是一个家仆之子，竟敢如此狂妄！”，说完，仿佛整个世界都静止了，魏无羡面色一变，心中怒火中烧，双手紧握，准备反击。就在这千钧一发之际，江厌离的声音传来：“阿羡！”

魏无羡不禁松了一口气，转眼望去，看到师姐向他招手。江厌离走上前，淡淡一笑：“阿羡，快到我身边来。”金夫人本想阻止她，但江厌离却向金子勋微微一礼，显得温和而坚定。

金子勋一时无言以对，心中虽然愤怒，却也无从反驳。江厌离温柔地说：“金公子，听说你说阿羡独占了百凤山三成的猎物，我实在不相信，从未听过这样的规矩。若是阿羡的确有不当之处，我替他向你道歉。”

魏无羡无奈地看着师姐，心里暗想：“你怎么能和他们讲大话，只会更糟。”

金子勋一阵窘迫，嘴角勉强露出笑容：“江姑娘说得对，不过这事真的很严重，他的确不守规矩。”

江厌离微微摇头，神色坚定：“我从未听过有哪条规矩说一个人猎得太多是错的。所以，若是他错了，金公子该反思一下。”她的话语一出口，那些围观的人都吓得哑口无言，金子勋面色渐渐阴沉下来，似乎找不到可反驳的理由。

“围猎就是围猎，跟家教有什么关系？”江厌离直言不讳，“阿羡是我云梦江氏的弟子，我对他的教育责任重大。金公子，你该为自己的言辞道歉。”

面对这样的直白，金子勋只得低头默不作声，心中无比憋屈。而江厌离则如同一阵春风，吹拂着场上的每一个人，带来舒适与和谐。

这时，金光瑶和蓝曦臣赶到了，他们见状，金夫人恼火地问：“你们这是怎么回事？”金光瑶急忙摆手：“母亲，我来看看情况。”蓝曦臣则皱起眉头：“大家请冷静，我会处理。”

金子勋转过身，冷冷说道：“整个百凤山的猎物都没了，五千多的参与者不就失去了机会？”他面露不甘，准备和其他人一起离开。

“不是吧，这次围猎简直就像一场闹剧！”如果不是江厌离的劝解，魏无羡早已火冒三丈。他只觉得这个家伙真的是太过分了。

就在众人纷纷欲走的时候，金子轩忽然一声大吼：“江姑娘！不是的！我不是勉强她，我是自己想要你来的！！！”

所有人都震惊地望向他，他慌慌张张地转身跑开，留下众人面面相觑。

魏无羡和江厌离对视一眼，忍不住笑出声来：“这家伙搞什么鬼！”

金夫人无奈叹气：“真是蠢货！我得去把他找回来。”说完，她带着一群修士追了上去。

“师姐，我们走吧。”魏无羡摆摆手，心中感慨。

“好啊，阿羡，走吧。”江厌离微微一笑，两人携手并肩，朝着远方走去，留下了一场纷争，和那一颗颗不安的心。 

风轻轻拂过，仿佛这场围猎虽未如愿，却为未来埋下了更多的希望和奇遇。
