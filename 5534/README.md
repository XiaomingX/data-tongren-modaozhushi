## 幻想修仙记：不夜天的纷争

魏无羡定了定神，眼前的场景似曾相识。回想当年，聂明玦接到情报，决定在阳泉进行一次突袭。赤锋尊向来勇猛无敌，谁知这次竟然遇上了岐山温氏的温若寒，他们的计划完全崩溃了。

“没想到，您也有今天这般狼狈。”孟瑶半跪在聂明玦身边，嘴角挂着一抹笑意。

聂明玦冷冷说：“滚开。”他正在愤怒，显然不想被这个调侃的家伙影响。

孟瑶微微一笑，带着戏谑的口气：“您还真的以为自己是河间王呢？看看周围，这里可是炎阳殿。”

旁边一名修士怒吼：“什么炎阳殿，不过一窝温狗的巢穴！”这话显然招惹了孟瑶。

“你说什么？”他眉头一挑，剑光一闪，那修士就毫无反应地倒下了。聂明玦怒火中烧，大吼道：“你！”

“温狗！有本事你也杀了我……”另一名修士叫嚣着，却被孟瑶毫不留情地一剑斩了去。看着两具白衣修士的尸体倒在地上，孟瑶轻蔑地问：“还有谁想说那个词的吗？”

聂明玦怒极反笑：“温狗。”他反正觉得落在温若寒手中就是死路一条，倒不如骂痛快些。

孟瑶不为所动，笑着打了个响指。身边一名温家修士急忙跪下来，双手举着一个长方形的盒子。孟瑶打开一看，赫然是聂明玦的佩刀——霸下！

“给我滚！”聂明玦顿时怒火中烧，但孟瑶却像故意挑衅似的，拿着刀摇了摇：“聂宗主，霸下可是曾在我手中走过不少场。我想问，温宗主这次要拍上几下，它才会断呢？”

聂明玦浑身的血都冲上了脑门，全身无比愤怒。他的父亲当年正是被温若寒的刀所害，他心中痛苦难忍，如今更难以忍受。

当年父亲在夜猎时，温若寒眨眼间就断了他老父的刀，结果被妖兽重伤，最后竟病死了。聂明玦见证了这一切，心中对岐山温氏的恨意积压至今，难以释怀。

“你在我面前扮什么？！”聂明玦怒吼，一掌劈出，孟瑶却稳稳后退，反手踹中他的胸口，聂明玦一个趔趄，吐出一口鲜血。

见状，温若寒坐镇的玉座有人站了起来。孟瑶立刻起身，脚下一踹，聂明玦就重重摔倒在地，胸腔的热血再次翻涌而出。魏无羡看得目瞪口呆，心想这场面真是戏剧化。

就在这千钧一发之际，聂明玦一掌拍出，正好突围，冲向温若寒。只见一名温家修士被他一把甩向玉座，正中温若寒，他怒吼：“无妨！”一掌击碎了那名修士的头颅。

聂明玦利剑出鞘，准备再战，孟瑶却拿着一根长剑，挑战他。“宗主当心！”

一阵笑声传来，源自温若寒，他似乎并不在意，再说听到聂明玦的战斗呼声，心里自有一番气定神闲。

魏无羡瞪大眼，突然看到身后一个身影，是蓝曦臣！“是什么情况，难道我也要上场吗？”

正当聂明玦要再施展剑术时，忽然间，他的手被一种重负击中，猝不及防，鲜血狂喷而出，一时之间，整个场面愈发紧张。

“这是在做什么？”魏无羡忍不住咋舌，“怎么这些人都跟瘟疫一样，见了我就要疯吗？”

……

经过一轮交手，场面混乱不堪。聂明玦伤重，孟瑶的计划似乎也未能达到预期。

而在这场绝望的斗争中，魏无羡突然发现，孟瑶不仅是个敌人，还是个值得信任的新朋友。尽管他手中握着利剑，心中却充满着奇妙的共鸣。此刻，他们已经不仅仅是敌人与朋友那么简单，而多了几分命运的纠缠。

最终，经过一番挣扎，他们都意识到：恩怨情仇，只是人世间的浮云，真正的武者应当超越这些，寻求心灵的解脱和强大。

魏无羡微微一笑，抬头望向天空，心中浮现出一幅新的画面：那是关于自由与梦想的未来。对于他们来说，或许，这才是真正的修仙之路。
