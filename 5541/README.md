```markdown
在一个古老的山谷里，蓝忘机站得笔挺，目光冷漠，淡淡地说：“没事。”

魏无羡则不像他那样冷淡，扭过身来，嬉皮笑脸地说道：“我们也算熟识吧？你这么冷漠，看都不看我一眼，你的腿真的没事？”

“不是熟。”蓝忘机简洁而坚决。

魏无羡一脸不甘，故意倒退着朝他走，想要让他看到自己的表情，“有事也别逞强啊！腿受伤了是吧？说说看，是什么时候的事？”

正当他想说“要不要我背你”，忽然一阵香风扑鼻而来。他回头一看，眼睛顿时亮了起来。几位少女正走过来，其中一个穿着淡粉色的外衫，外面罩着薄薄的纱衣，微风一吹，衣袖飘逸，背影引人注目。

一个少女兴奋地说：“绵绵，你这个香囊真是太棒了！自从带上它，蚊虫都不来了，而且闻起来特别好，真的感觉神清气爽！”

被称作绵绵的少女声音软软的：“香囊里装的是一些药材，效果不错。我这里还有几份，谁需要？”

魏无羡立刻蹭了过去，调皮地说：“绵绵，给我留一个！”

那少女愣了一下，立刻转过头，眉头微蹙：“你是谁？为什么也叫我绵绵？”

魏无羡笑着解释：“我听她们都叫你这个名字，以为就是你的名字呢。怎么，不是吗？”

蓝忘机在侧边冷眼旁观，江澄见他又开窍了，忍不住翻了个白眼。

“你不许这样叫我！”绵绵羞红了脸，表情生气。

“为什么不许？这样吧，你告诉我你的名字，我就不叫你绵绵，怎么样？”魏无羡继续卖弄自己的幽默。

“问别人名字之前，自己不先报上名字，凭什么告诉你我的名字？”绵绵反问。

魏无羡自信回答：“我的名字没什么好隐瞒的！我叫远道。”

绵绵重复了几遍“远道”，心中却想不起来这名字出自哪个世家，但看到他嘴角那坏坏的笑容，内心略感疑惑。  

就在这时，蓝忘机低声说道：“玩弄字眼。”

绵绵瞬间恍若被点醒：“谁思你了！你太不要脸！”

几名少女都忍不住笑了起来，有的说：“魏无羡，你真是太讨厌了！”还有的说：“没见过你这么无耻的！”

绵绵拉着她们离开，心里那些小心思被烧得一干二净，“走，走！不准你们跟他说。”

魏无羡在后面对她们喊：“走可以，香囊给我留一个嘛！不给我？那我就找别人问你什么名字，总有人告诉我……”

话没说完，突然有一个香囊飞过来，正砸在魏无羡的胸口，他“哎哟”一声做出心痛的样子，香囊的带子还缠在他的手指上，走回蓝忘机身边，边转边笑。

看到蓝忘机的脸色更冷，魏无羡问：“怎么了？又这样盯着我。对了，上次咱们说到哪了？我背你吧？”

蓝忘机静静地看着他，冷冷说道：“你对所有人都这样吗？这就是你的风格？”

魏无羡思索了一下，认真回答：“好像是吧。”

蓝忘机低下头，半晌，只吐出两个字：“轻狂！”他咬着牙说出这两个字，明显难受得很，便继续加快步伐。

魏无羡看他这幅样子，又有些担心，“别急啊，你的腿还没好呢，我真的可以背你。”

江澄则对魏无羡不屑一顾：“你无聊吗？蓝湛是个什么样的人，你不清楚吗？”

魏无羡不以为然：“他今天脸色更糟，腿上的伤也不清楚怎么办。再说，这事儿我还挺在意的，我去背他！”

听着这话，江澄忍不住紧张：“你逗我呢吧！他会同意吗？看他那态度！”

“他讨厌我又怎么样？我借他背着他而已，他能掐死我？！”魏无羡得意洋洋。

江澄皱眉：“顾好自己吧，我们还得去找温晁那个家伙呢！”

这时，突然有个温氏家仆过来，斥责：“别窃窃私语，注意一点！”

接着，王灵娇走了过来，她是温晁的贴身侍女，人人都知道她跟温晁的关系。她若无其事地指着一名少女道：“就她吧！”

那少女正是绵绵，听到自己被点到，立刻被吓得面白，连退几步。魏无羡心里一惊，拉住江澄道：“她不会被他们绑了吧？”

温晁嘴角勾起一抹坏笑：“哎，捉了她，放血引兽，肯定有效！”

眨眼间，局势急转直下，绵绵恐慌不已，想逃却无处可去。魏无羡看着，心里顿时紧张，不想再观望下去了。蓝忘机也终于出手，曾经让他们追逐的恶梦，现在却得面对温家的围攻。

魏无羡试图制止局面：“别动，别伤到她！”

温晁气愤，吼着：“你们这是要造反吗！”

正当局面紧张时，谁也没想到，地面突然颤动，魏无羡警觉地问：“江澄，地动了吗？”

江澄摇头，但魏无羡觉得似乎是石岛在晃荡。意识到自己站在一只庞然大物的背壳上，他的心脏狂跳不止，似乎并不简单。
```
