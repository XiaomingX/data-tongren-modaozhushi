```markdown
在一个雷雨交加的夜晚，江澄被一剑刺中，虽然没有生命危险，但他痛得动不了，灵力也暂时无法运用。他偏偏不喜欢别人扶他，抓住金凌的衣袖，冷冷地说：“快滚！”

金凌感到心虚，他知道自己乱跑惹怒了舅舅，不敢反驳。但就在此时，远处有一只黑鬃灵犬狂吠，突然传来一声哀叫，金凌吓了个激灵，想起金光瑶之前的警告，大喊：“仙子快跑，他们要杀你！”

不久之后，苏涉冒雨跑了过来，他的表情愤怒得快要爆炸。金光瑶白了他一眼：“怎么没杀死那家伙？”

苏涉咬牙切齿：“属下无能。那狗居然一看到危险就跑得比谁都快！”

金光瑶摇摇头，语气沉重：“再多的狗也只能叫唤，快点解决问题。”

“这群无用的废物！我去催催。”苏涉怒火中烧。

金凌松了口气，看到江澄坐在地上，脸色苍白，不禁有些担心。他犹豫片刻，对蓝忘机说：“含光君，还有蒲团吗？”

那时他们坐的四个蒲团都是蓝忘机找来的，而这观音殿里就只剩下几个。沉默片刻，蓝忘机站起来，把他自己的蒲团推了过去。

金凌忙道：“谢谢！不用了，我还是想坐我自己的……” 

蓝忘机则淡然道：“不必。”

于是，蓝忘机就在魏无羡身边坐了下来。他们两个在同一个蒲团上坐得一本正经，似乎完全不觉得挤。看到空间被让出来，金凌犹豫着拖着江澄过去，试图让舅舅靠着自己，忍痛坐下后，江澄眼神低垂，不知道在想些什么。

这时，殿后传来一声兴奋的呼喊：“宗主！找到了！露出一角了！”

金光瑶的脸色一下子缓和了，快步走回去：“抓紧继续！小心点，时间不多了。”

电闪雷鸣，魏无羡和蓝忘机坐在一起，金凌把自己的蒲团也拖了过去。四周静得出奇，谁都没开口。

金凌心里奇怪，想让大家聊聊，他四处乱瞅，突然说：“舅舅，多亏你刚才挡住那根琴弦，不然就糟了。”

江澄脸色阴沉，道：“你给我闭嘴！”

要不是他情绪不稳，金光瑶也不会趁机偷袭。而且，他们其实完全可以躲开那根琴弦。金凌这话说得有点刻意，大家的气氛更加尴尬。

被训斥后，金凌闭上了嘴。江澄不动，魏无羡也是沉默。

如果是在过去，魏无羡一定会用尖酸的语气讥讽江澄几句，但现在，他反而默默想起金光瑶说过的话，似乎明白了江澄的心情。

就在这时，蓝忘机轻轻拍了拍江澄背部，魏无羡轻声问：“……你知道吗？”

蓝忘机缓缓点头。

“……温宁。”魏无羡几乎是低声呢喃。

原来温宁在手里的东西，现在在江澄手里。而他们离开莲花坞的路上，温宁却对此闭口不谈。

“他什么时候说的？”魏无羡问。

“你失去意识的时候。”蓝忘机回答。

魏无羡心中一紧：“我们就这样离开莲花坞的？”

蓝忘机道：“他很内疚。”

“我一直叮嘱他不要说的！”魏无羡无奈。

突然，江澄插话：“不要什么？”

魏无羡和蓝忘机对视，看到江澄一手捂着伤口，冷冷地道：“魏无羡，你真伟大。做好事还低调，真让人感动，我是不是该跪下来感谢你？”

江澄的话毫不客气，蓝忘机皱起了眉。金凌紧张地挡在江澄前面，生怕蓝忘机会发怒。

魏无羡沉默着，心情也有些复杂。一开始不想让江澄知道太多事情，是想保护他，没想到如今还是得面对。

江澄眼中的愤怒越发浓重：“凭什么，魏无羡，你凭什么？”

“我们江家给了你多少？明明我才是儿子，应该是继承人，这么多年我处处被你压着！”

金凌面色一白，魏无羡也沉默了。江澄继续怒吼，情绪越来越激动：“你是什么？凭什么我反而觉得对不起你？”

金凌慌忙说：“舅舅，你别这样！”

江澄不顾他，愤怒释放：“那又如何？我恨你，我不能恨你吗？”

此时，蓝忘机突然站起身，金凌紧张地挡在江澄前面：“含光君！我舅舅受伤了……”

但江澄推开了金凌，愤怒的情绪在他的心中翻涌着，怒火几乎要将他吞噬。

就在此时，江澄流下了泪水，满脸荒唐与痛苦：“凭什么……你凭什么不告诉我！”

他捏紧拳头，像是在内心挣扎，最终却无力地砸向地面。这一刻，他知道，他无法完全仇恨魏无羡。

魏无羡不知道如何回应。当初是为了不让江澄如今日常生活受到困扰，才决定保持沉默。可现在他忍不住思考，是否这样做是对的。

江澄哭泣的声音萦绕在耳边，他心里在挣扎：“你说过，永远支持我！你居然食言！”

“对不起。”魏无羡终究说出了这句简单的话。

江澄摇了摇头，脸埋在手掌里，半晌，突然笑了，嘴角透出讥讽，他闷声道：“还想对我说对不起？我可是多金贵的人呢。”
```
